# Cursor Skills for StockSignal

This directory contains AI-assisted development skills ported from the Pirouette sibling project.

## Available Skills

| Skill | Purpose | Trigger |
|-------|---------|---------|
| **project-scaffolder** | Rapid SaaS project setup | "scaffold new project" |
| **skill-import-assistant** | Import code from sibling projects | "import skills from [project]" |
| **scaling-calculator** | Infrastructure cost projections | "calculate scaling costs" |
| **email-touchpoint-mapper** | Plan email touchpoints | "plan email touchpoints" |
| **prd-progress-tracker** | Track PRD alignment | "check PRD alignment" |

## How to Use

These skills are automatically available to Cursor AI. Simply use the trigger phrases in your conversations.

For detailed documentation of each skill, see the individual `.md` files or the original Pirouette skills at:
`/Users/tomeldridge/pirouette/.cursor/skills/`

## Adding New Skills

1. Create a new `.md` file in this directory
2. Follow the standard format:
   - Purpose
   - Origin
   - Applicability
   - Trigger Phrases
   - Protocol/Usage
3. Update this README
